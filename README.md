# Professional Resume Builder

A sleek, ATS-friendly, and modern resume builder built with HTML, Tailwind CSS, and Vanilla JavaScript.

## Features
- **Modern Design**: Vibrant colors, sleek typography, and responsive layout.
- **ATS-Friendly**: Structured for optimal performance with Applicant Tracking Systems.
- **Interactive**: Download as HTML or Save as PDF functionality built-in.
- **Micro-animations**: Subtle hover effects and transitions for a premium feel.
- **Print Optimized**: Custom print styles specifically for PDF generation.

## Technologies Used
- HTML5 / Semantic HTML
- CSS3 (Vanilla & Tailwind CSS)
- JavaScript (ES6+)
- FontAwesome Icons
- Google Fonts (Plus Jakarta Sans)

## How to Use
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Use the "Download HTML" or "Save as PDF" buttons to export your resume.
